# Thineth Shalinda - Developer Portfolio

A premium, interactive developer portfolio built to showcase full-stack projects, responsive UI designs, and technical skills. It features a unique command-line terminal emulator, micro-animations, and a next-level glassmorphism design system.

![Portfolio Preview](./public/jack-sparrow.jpg)

## 🚀 Key Features

- **Interactive Terminal Emulator**: A fully functional CLI component allowing visitors to run commands (`help`, `skills`, `projects`, `about`, `contact`) to explore the portfolio data.
- **Glassmorphism & Micro-animations**: State-of-the-art UI utilizing floating animations, glowing blobs, and frosted glass panels for a premium aesthetic.
- **Dynamic Projects Gallery**: Filterable project showcase featuring detailed modals with system architectures and tech stacks.
- **Responsive Architecture**: Fully responsive layout that ensures an optimal viewing experience across both mobile and desktop platforms.
- **Live Tech Stack display**: Visually appealing categorization of skills, programming languages, and tools.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: TypeScript / React
- **Icons & Fonts**: Next/font (Geist), Custom SVG icons

## 💻 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Thineth388/your-repo-name.git
   ```

2. **Navigate into the directory**
   ```bash
   cd your-repo-name
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
